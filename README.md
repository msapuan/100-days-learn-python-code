## About

This repository represents my dedicated 100-day learning challenge to master Python programming. Each day's code showcases practical implementations of new concepts, best practices, and evolving problem-solving skills.

**Purpose:**
- Document my learning progression systematically
- Build a portfolio of Python implementations
- Reinforce concepts through consistent daily practice
- Create a resource for fellow Python learners

## Learning Objectives

Throughout this 100-day journey, I'm focusing on:

- **Fundamentals** (Days 1-20): Variables, data types, operators, control flow, functions
- **Data Structures** (Days 21-35): Lists, tuples, dictionaries, sets, comprehensions
- **Object-Oriented Programming** (Days 36-50): Classes, inheritance, polymorphism, encapsulation
- **Advanced Topics** (Days 51-75): File I/O, exception handling, decorators, generators
- **Practical Applications** (Days 76-100): APIs, databases, web frameworks, real-world projects

## Getting Started

### Prerequisites

- Python 3.7 or higher
- Git for version control
- A text editor or IDE (VS Code, PyCharm, etc.)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/100-days-python.git
   cd 100-days-python
   ```

2. **Verify Python installation:**
   ```bash
   python --version
   ```

### Running the Code

To run any day's script:

```bash
python day01.py
```

Each script is executable individually without external dependencies unless otherwise specified.
